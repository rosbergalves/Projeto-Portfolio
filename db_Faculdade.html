Banco de Dados db_Faculdade

Sobre o Projeto

O projeto db_Faculdade foi desenvolvido com o objetivo de criar um banco de dados estruturado para a gestão acadêmica de uma faculdade, contemplando alunos, professores, cursos e disciplinas.

Detalhes do Desenvolvimento

Este banco de dados foi modelado seguindo as melhores práticas de normalização e otimização, garantindo consistência e integridade nos dados armazenados.

  • Levantamento de requisitos e análise de necessidades
  • Modelagem conceitual e lógica
  • Implementação no SQL Server
  • Testes e validação de dados
